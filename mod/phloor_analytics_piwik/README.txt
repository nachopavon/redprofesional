/**
 * Description
 */
Enables website tracking with Piwik on your Elgg site.

Only tested with Piwik 1.6

Settings:
'enable_tracking': Enables loading the Piwik scripts
'site_id': Piwik Site ID for the site
'path_to_piwik': Path to the Piwik Installation

See also: http://piwik.org/

/**
 * Languages
 */
English
German

/**
 * Admin Settings
 */
Configuration of the following settings:
* 'enable_tracking': Enables loading the Piwik scripts
* 'site_id': Piwik Site ID for the site
* 'path_to_piwik': Path to the Piwik Installation
