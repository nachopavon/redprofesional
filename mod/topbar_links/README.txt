This is a Elgg plugin that allow to extend the topbar and set new links

REQUIREMENTS
============

 - Elgg version 1.8.

INSTALLATION
============

 - Copy the ``topbar_links`` directory to the mod directory in an Elgg
   installation.

USAGE
=====

 - The configuration params of this plugins are in settings.php. There
   you can create as many links as you want in the topbar section.
   Also you can choose if you want the default elgg topbar links or not.

 - In /topbar_links you will find a view that renders the topbar without
   the elgg links (just with the links added in settings.php)
